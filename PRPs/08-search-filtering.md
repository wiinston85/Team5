# PRP 08: Search and Filtering

## Feature Overview
Enable fast, combined search and filtering for todo discovery.

## Technical Requirements
- Debounced search at 300ms.
- Search targets title and tag names.
- Combined AND logic across search, priority, and tag filters.

## UI Requirements
- Search input.
- Priority and tag filters.
- Clear all filters action.

## Acceptance Criteria
- Case-insensitive search works.
- Combined filters work.
- Empty state shown when no matches.

## Testing Requirements
- E2E search and combined filter coverage.
